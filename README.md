# Emxc

a Tesla REST client for the MEXC Exchange

## Installation

The package can be installed by adding `emxc` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:emxc, "~> 0.0.3"}
  ]
end
```
